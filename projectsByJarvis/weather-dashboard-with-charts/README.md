# weather-dashboard-with-charts

A modern and responsive weather dashboard displaying current weather conditions, forecasts, and historical data visualized in interactive charts.

## Features
- Current weather conditions display (temperature, humidity, wind speed, pressure, description, icon)
- 5-day weather forecast with daily summaries
- Hourly weather forecast for the next 24 hours
- Interactive charts for temperature, humidity, and wind speed over time
- Location search functionality with autocomplete
- Geolocation support to automatically detect user's location
- Responsive design for optimal viewing on various devices (desktop, tablet, mobile)
- User-friendly interface with clear data visualization
- Error handling for API requests and location services
- Data persistence (using local storage) for user preferences (location)
- Units selection (Celsius/Fahrenheit, m/s/kmh/mph)
- Accessibility considerations (ARIA attributes, keyboard navigation)

## Tech Stack
- HTML5
- CSS3 (INCLUDING FLEXBOX AND/OR GRID FOR LAYOUT)
- JAVASCRIPT (ES6+)
- REACT (FOR COMPONENT-BASED ARCHITECTURE AND EFFICIENT UI UPDATES)

## Architecture
The application follows a component-based architecture using React. The `App` component is the main container and renders other components such as `CurrentWeather`, `Forecast`, `HourlyForecast`, `WeatherChart`, and `LocationSearch`.  The `weatherService` module handles API calls to a weather API (e.g., OpenWeatherMap).  Data fetched from the API is passed down as props to the relevant components.  Chart.js is used to create interactive charts based on the weather data.  The application uses a unidirectional data flow.  User input (e.g., location search) triggers API calls, updates the application state, and re-renders the UI.  Local storage is used to persist user preferences (location) across sessions.  Webpack bundles all the modules and assets into a single bundle for deployment.
